# Задание 1

## Задача 1

### Текст задачи
Разработать класс с двумя логическими полями. Создать конструктор копирования. Разработать метод, вычисляющий дизъюнкцию полей. Перегрузить метод ToString() для формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и другие методы базового и дочернего классов.

### Алгоритм решения
1\. создаётся класс Device

2\. в классе объявляются два логических поля isOn и hasPower

3\. реализуется конструктор с параметрами для инициализации полей

4\. реализуется конструктор копирования, который копирует значения полей из другого объекта

5\. реализуется метод CanWork

6\. в методе выполняется логическая операция ИЛИ над полями isOn и hasPower

7\. если хотя бы одно из значений истинно, метод возвращает true

8\. если оба значения ложны, метод возвращает false

9\. переопределяется метод ToString для формирования строки с информацией о состоянии устройства

10\. создаётся дочерний класс SmartDevice, который наследуется от класса Device

11\. в дочернем классе объявляется дополнительное логическое поле isConnectedToWifi

12\. реализуется конструктор с параметрами, который вызывает конструктор базового класса

13\. в конструкторе инициализируется поле isConnectedToWifi

14\. реализуется конструктор копирования, который вызывает конструктор копирования базового класса

15\. в конструкторе копируется значение поля isConnectedToWifi

16\. реализуется метод IsOnline

17\. метод возвращает значение поля isConnectedToWifi

18\. реализуется метод IsSmartActive

19\. в методе вызывается метод CanWork базового класса

20\. результат объединяется с полем isConnectedToWifi с помощью логического И

21\. если оба условия истинны, метод возвращает true

22\. иначе возвращается false

23\. реализуется метод ConnectToWifi

24\. в методе устанавливается значение поля isConnectedToWifi равным true

25\. переопределяется метод ToString

26\. в методе вызывается ToString базового класса и добавляется информация о Wi-Fi

27\. в методе Main создаются объекты классов и выполняется вызов их методов для проверки работы программы

# Задание 2

## Задача 1

### Текст задачи
Определить пересекаются ли заданные отрезки. Результат должен быть типа bool.

### Алгоритм решения
1\. заданы два отрезка с координатами x и y

2\. для первого отрезка определяется начало как минимальное из x и y

3\. для первого отрезка определяется конец как максимальное из x и y

4\. для второго отрезка определяется начало как минимальное из его x и y

5\. для второго отрезка определяется конец как максимальное из его x и y

6\. проверяется условие: конец первого отрезка меньше начала второго

7\. проверяется условие: конец второго отрезка меньше начала первого

8\. если выполняется хотя бы одно из условий, отрезки не пересекаются

9\. если оба условия ложны, отрезки пересекаются

10\. результат возвращается в виде значения true или false

# Задание 3

## Задача 1

### Текст задачи
Унарные операции:
! вычислить длину диапазона, результатом должно быть вещественное
число

++ расширить отрезок на 1 вправо и влево

Операции приведения типа:

int (неявная) – результатом является целая часть координаты х;

double(явная) – результатом является координата y.

Бинарные операции:

\- LineSegment целое число (левосторонняя операция, уменьшается
координата х);

\- целое число LineSegment (правосторонняя операция, уменьшается
координата y);

< LineSegment d – результат равен true, если отрезки пересекаются, и false–
в противном случае.

### Алгоритм решения
1\. создаётся объект класса LineSegment с координатами x и y

2\. для унарной операции ! вычисляется длина отрезка

3\. из координаты y вычитается координата x

4\. берётся модуль разности

5\. результат возвращается как вещественное число

6\. для унарной операции ++ уменьшается координата x на 1

7\. увеличивается координата y на 1

8\. возвращается изменённый отрезок

9\. для неявного преобразования в int берётся координата x

10\. отбрасывается дробная часть

11\. возвращается целое значение

12\. для явного преобразования в double возвращается координата y

13\. для операции LineSegment - int уменьшается координата x на заданное число

14\. возвращается изменённый отрезок

15\. для операции int - LineSegment уменьшается координата y на заданное число

16\. возвращается изменённый отрезок

17\. для операции < вызывается метод проверки пересечения отрезков

18\. если отрезки пересекаются возвращается true

19\. если не пересекаются возвращается false
